### break

> 【华南理工大学2018研】编程求100～200间的全部素数。
>
> 【答案】
> 素数意思是只能被1和本身整除，因此将1到本身之间的数做除数，进行求余，如果余数为0，则不是素数，否则是素数。根据经验，假设所要判断的数为n，则一般只需要判断1到根号n之间的数即可，具体程序如下：
>
> ```c
> #include<stdio.h>
> #include<math.h>
> 
> void main() {
>     int i, j, count;
>     for (i = 100; i <= 200; i++) {
>         count = 0;
>         for (j = 2; j < sqrt(i); j++) {
>             if (i % j == 0)
>                 count++;
>         }
>         if (count == 0)
>             printf("%d 是素数!\n", i);
>     }
> } 
> ```
